# Changelog

## 0.1.1 - 2026-06-16

- Added validator contract checks for scaffold naming, runtime trigger thresholds, and protected public support surfaces.
- Rejected overlong normalized skill names instead of silently truncating them.
- Strengthened runtime skill validation by requiring more specific trigger descriptions.
- Expanded structure validation across localized READMEs, docs, templates, examples, forward prompts, references, and scripts.
- Added concrete expected checks to forward prompts and recorded the coverage review.
- Clarified install smoke-test behavior and validation documentation.
- Completed localized README content for German, Spanish, French, and Brazilian Portuguese.

## 0.1.0 - 2026-06-14

- Initial public release of AI Skill Create.
- Added self-contained Codex plugin package.
- Added `ai-skill-create` skill with references, scripts, and UI metadata.
- Added Windows-first install dry-run flow.
- Added validation scripts for structure, metadata, links, syntax, install, and secrets.
- Added public documentation, examples, GitHub templates, and CI workflows.
